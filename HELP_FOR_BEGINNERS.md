## 🤔 СНАЧАЛА ОТВЕТЬ НА ВОПРОСЫ:

### Вопрос 1: У тебя установлен PHP?
**Как проверить:**
1. Откройте Windows PowerShell (нажмите Win, напишите PowerShell)
2. Введите: `php --version`
3. Если видишь версию (например "PHP 8.2.0") - **ДА, установлен**
4. Если видишь ошибку - **НЕТ, не установлен**

### Вопрос 2: У тебя установлена MySQL?
**Как проверить:**
1. Откройте Windows PowerShell
2. Введите: `mysql --version`
3. Если видишь версию - **ДА, установлена**
4. Если видишь ошибку - **НЕТ, не установлена**

## 📋 ВЫБЕРИТЕ СВОЙ СЦЕНАРИЙ:

### ✅ Сценарий 1: У меня ЕСТЬ PHP и MySQL

**Что делать:**
1. Откройте PowerShell
2. Введите:
   ```bash
   mysql -u root -p
   ```
   (если спросит пароль - просто нажмите Enter или введите ваш пароль)

3. Скопируйте-вставьте (все сразу):
   ```sql
   CREATE DATABASE cms;
   USE cms;
   SOURCE c:/Users/SystemX/Desktop/cms/database/schema.sql;
   EXIT;
   ```

4. В папке `cms` откройте PowerShell (Shift + правый клик)
5. Введите:
   ```bash
   php -S localhost:8000 -t public
   ```

6. Откройте браузер: http://localhost:8000

**Вход:**

- Пароль: admin123

✅ **Готово!**

### ❌ Сценарий 2: У меня НЕТ PHP и MySQL

**Что делать:**

#### Способ A: Установить XAMPP (проще!)

1. **Скачайте XAMPP:**
   - Откройте: https://www.apachefriends.org/
   - Нажмите **Download XAMPP**
   - Скачайте Windows версию
   
2. **Установите:**
   - Двойной клик на загруженный файл
   - На всех вопросах нажимайте "Next"
   - Установите в `C:\xampp\`
   
3. **Запустите XAMPP:**
   - Откройте `C:\xampp\xampp-control.exe`
   - Нажмите **Start** на Apache
   - Нажмите **Start** на MySQL
   
4. **Скопируйте CMS:**
   - Скопируйте папку `cms` в `C:\xampp\htdocs\`
   
5. **Создайте БД:**
   - Откройте браузер: http://localhost/phpmyadmin
   - Нажмите вкладку **SQL**
   - Скопируйте содержимое файла `C:\xampp\htdocs\cms\database\schema.sql`
   - Вставьте в текстовое поле
   - Нажмите **Выполнить**
   
6. **Откройте CMS:**
   - http://localhost/cms/public
   
**Вход:**
- Email: admin@example.com
- Пароль: admin123

✅ **Готово!**

## 💾 ЕСЛИ БАЗА ДАННЫХ НЕ СОЗДАЕТСЯ

### Через XAMPP (phpMyAdmin):

1. Откройте: http://localhost/phpmyadmin
2. Слева нажмите на **"Создать"** или **"New"**
3. В поле "Database name" введите: `cms`
4. Нажмите **Create**
5. Теперь нажмите вкладку **SQL** (вверху)
6. Скопируйте весь код из `database/schema.sql` и вставьте
7. Нажмите **Выполнить**

### Через MySQL (CommandLine):

```bash
mysql -u root -p
CREATE DATABASE cms;
USE cms;
SOURCE c:/Users/SystemX/Desktop/cms/database/schema.sql;
EXIT;
```

```php

```

(Если пароля нет - оставьте пустую строку '')
