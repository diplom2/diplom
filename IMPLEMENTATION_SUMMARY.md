## 📋 Таблицы БД (7 таблиц)

1. **users** - Пользователи системы
   - id, name, email, password, role, status, created_at, updated_at
   - Index: email, role

2. **categories** - Категории постов (иерархическая структура)
   - id, name, slug, description, parent_id, created_at, updated_at
   - Index: slug, parent_id

3. **posts** - Блог-посты
   - id, title, slug, content, excerpt, category_id, author_id, status, featured_image_id, created_at, updated_at, published_at
   - Index: slug, status, author_id, category_id

4. **media** - Загруженные медиафайлы
   - id, filename, original_name, mime_type, size, uploaded_by, uploaded_at
   - Index: mime_type, uploaded_by

5. **comments** - Комментарии к постам
   - id, post_id, user_id, author_name, author_email, content, status, created_at, updated_at
   - Index: post_id, status, user_id

6. **settings** - Параметры системы (ключ-значение)
   - id, key, value, created_at, updated_at
   - Index: key

7. **audit_logs** - Логирование изменений (аудит)
   - id, user_id, action, model, model_id, changes, ip_address, created_at
   - Index: user_id, created_at

## 🎨 Интерфейсы

### Админ-панель
- Отзывная сайдбар навигация с меню, зависящим от роли
- Верхняя полоса с информацией пользователя
- Карточки статистики (количество постов, пользователей, комментариев)
- Таблицы для управления данными
- Формы с валидацией клиента и сервера

**Роли админ-панели:**
- **Администратор** - Полный доступ ко всему
- **Редактор** - Посты, категории, медиа, комментарии
- **Автор** - Только свои черновики и медиа

### Фронтенд (публичная часть)
- Главная страница со списком постов
- Страницы отдельных постов с комментариями
- Страницы категорий
- Страница поиска
- Мобильный дизайн первый подход (responsive)

## ⚙️ Установка и использование

### Требования
- PHP 7.4+
- MySQL 5.7+
- Apache или Nginx с mod_rewrite
- Composer (опционально)

### Быстрая установка

1. **Windows:**
   ```
   install.bat
   ```

2. **Linux/Mac:**
   ```
   bash install.sh
   ```

3. **Вручную:**
   - Скопировать файлы на сервер
   - Создать БД и импортировать schema.sql
   - Скопировать config/database.php.example → config/database.php
   - Изменить учетные данные БД
   - Импортировать администратора из seed_admin.sql
   - Изменить права на папок storage и uploads

### Вход в систему
- URL: http://example.com/login
- Email: admin@example.com
- Пароль: admin123

### Изменение пароля
После первого входа **ОБЯЗАТЕЛЬНО** измените пароль администратора.
