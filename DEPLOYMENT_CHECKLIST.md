## ⚙️ Критически важные переменные окружения

```
DB_HOST=*****.mysql.render.com          # Из Connection String БД
DB_PORT=3306                             # Обычно 3306
DB_NAME=cms_database                     # Имя БД (по умолчанию)
DB_USER=cms_admin                        # Пользователь БД (по умолчанию)
DB_PASSWORD=your_password_here           # Пароль (ВАЖНО: скопировать из Connection String)
```

❗ **Особенно проверьте:**
- `DB_HOST` - скопировать из Connection String
- `DB_PASSWORD` - НЕ потеряйте, сохраните безопасно
